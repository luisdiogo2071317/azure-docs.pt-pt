Mantenha as seguintes considerações de segurança em mente quando utiliza a StorSimple Cloud Appliance:

* A aplicação da cloud é protegida através da sua subscrição do Microsoft Azure. Isto significa que se estiver a utilizar a aplicação da cloud e a subscrição do Azure for comprometida, os dados armazenados na sua aplicação da cloud são suscetíveis.
* A chave pública do certificado utilizado para encriptar dados armazenados no StorSimple é disponibilizada no portal do Azure de forma segura e a chave privada é mantida com a StorSimple Cloud Appliance. Na StorSimple Cloud Appliance, ambas as chaves públicas e privadas estão armazenadas no Azure.
* A aplicação da cloud está alojada no datacenter do Microsoft Azure.
