## <a name="create-an-azure-storage-account"></a>Criar uma conta do Storage do Azure

A forma mais fácil de criar a primeira conta de armazenamento do Azure é com o [portal do Azure](https://portal.azure.com). Para saber mais, veja [Criar uma conta de armazenamento](../articles/storage/common/storage-quickstart-create-account.md).

Também pode utilizar o [Azure PowerShell](../articles/storage/common/storage-powershell-guide-full.md), a [CLI do Azure](../articles/storage/common/storage-azure-cli.md) ou o [Fornecedor de Recursos do Armazenamento do Azure para .NET](https://azure.microsoft.com/resources/samples/storage-dotnet-resource-provider-getting-started/).

Se preferir não criar uma conta de armazenamento no Azure neste momento, também pode utilizar o emulador de armazenamento do Azure para executar e testar seu código num ambiente local. Para obter mais informações, veja [Utilizar o Emulador de Armazenamento do Azure para Programação e Teste](../articles/storage/common/storage-use-emulator.md).
