# <a name="team-data-science-process-for-data-scientists"></a>Processo de ciência de dados de equipa para cientistas de dados

Este artigo fornece orientações para um conjunto de objetivos que são normalmente utilizadas para implementar as soluções de ciência de dados abrangente com tecnologias do Azure. É guiado pelos:

- Noções básicas sobre uma carga de trabalho de análise
- usando o Team Data Science Process
- com o Azure Machine Learning
- a base da transferência de dados e armazenamento
- fornecer documentação de origem de dados
- com as ferramentas para processamento de análise

Esses materiais de treinamento relacionados com o processo de ciência de dados de equipa (TDSP) e Microsoft e software open-source e kits de ferramentas, o que são úteis para a previsão, executar e fornecer soluções de ciência de dados.
