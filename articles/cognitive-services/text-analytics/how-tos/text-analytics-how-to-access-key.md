# <a name="how-to-find-endpoints-and-access-keys-for-the-text-analytics-cognitive-service"></a>Como encontrar pontos de extremidade e acesso chaves para o serviço cognitivos de análise de texto

Quando se inscreve para análise de texto, obtém uma chave de acesso personalizado exclusivo à sua subscrição. Esta chave é necessária em cada chamada para a API de análise de texto. Se ainda não o tiver [inscreveu](text-analytics-how-to-signup.md), faça-o agora para obter a chave. 

Se precisar de ajuda a localizar a chave, ou para determinar se a sua subscrição já tem de análise de texto, utilize as instruções seguintes para obter as informações necessárias. 

## <a name="find-your-service-endpoint-and-access-key"></a>Localizar a chave de acesso e de ponto final de serviço

1. Inicie sessão no [portal do Azure](https://portal.azure.com).

2. No painel de navegação esquerdo, selecione **todos os serviços**.

3. No filtro, escreva *dos serviços cognitivos*. Se a sua subscrição tem de análise de texto, aparece na lista, com um tipo de API do **API de análise de texto**.

4. Clique na ligação para abrir o painel de serviço. Agora pode obter uma chave a partir **recursos > chaves** ou ao clicar em **Mostrar chaves de acesso** no painel Essentials. O ponto final também está visível no painel Essentials.
