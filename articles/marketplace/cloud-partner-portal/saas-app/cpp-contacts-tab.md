# <a name="saas-application-contacts-tab"></a>Separador de contactos de aplicação SaaS

Este artigo descreve como utilizar o separador de contactos para identificar os contactos para os clientes que estão a utilizar a oferta.

## <a name="identify-contacts"></a>Identificar os contactos

Utilize a seguinte captura de ecrã como guia para fornecer as informações necessárias. Todos os campos com marcados com um asterisco (*) deve ser concluído.

![Formulário de contactos de suporte e de engenharia](./media/saas-offer-contacts.png)

1. Em contacto de engenharia, fornecem informações para os seguintes campos:

   - Dê um nome, introduza o nome do contacto de engenharia para a sua aplicação. Este contacto receberá comunicações técnicas da Microsoft.
   - E-mail - introduza o endereço de e-mail do contacto de engenharia.
   - Telefone - introduza o número de telefone de contacto de engenharia. Notações de número de telefone ISO são suportadas. Para obter mais informações, consulte [E.123](https://en.wikipedia.org/wiki/E.123)
  
2. Em contacto de suporte, forneça informações para os seguintes campos:

   - Dê um nome, introduza o nome do contacto de suporte para a sua aplicação. Este contacto receberá as comunicações relacionadas ao suporte da Microsoft.
   - E-mail - introduza o endereço de e-mail de contacto de suporte para a sua aplicação.
   - Telefone - introduza o número de telefone de contacto de suporte. Notações de número de telefone ISO são suportadas. Para obter mais informações, consulte [E.123](https://en.wikipedia.org/wiki/E.123)
   - URL de suporte – introduza o URL para sua página de suporte.

3. Selecione **guardar** quando terminar de formulários.
