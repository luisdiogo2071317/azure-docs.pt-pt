# <a name="create-azure-marketplace-and-appsource-offers"></a>Criar ofertas do Azure Marketplace e AppSource

Um objetivo essencial da Cloud Partner Portal é permitir que os editores a criar (e, em seguida, publicar) oferece o Microsoft Azure e mercados de AppSource.  Esta operação sempre começa com a seleção do tipo de oferta pretendido do [New oferecem menu](../portal-tour/cpp-new-offer-menu.md).  Em resposta, o adequado **nova oferta** é apresentada a página para esse tipo de oferta.  Por exemplo, a imagem seguinte mostra a predefinição **nova oferta** página para um tipo de aplicação do Azure.

![Nova página de padrão de oferta](./media/new-offer-page.png)

Há duas seleções de separador disponíveis na barra de menus horizontal apresentada na parte superior desta página: 
- **Editor** separador – permite a entrada de informações e o carregamento de recursos para a nova instância de oferta.  Este separador é apresentado por predefinição.
- **Estado** separador - fornece o estado de publicação e apresenta uma lista de problemas de validação e de revisão. 

Quando cria uma oferta, utilize o **Editor** separador para introduzir informações sobre essa oferta. 

## <a name="editing-operations"></a>Operações de edição

Barra de ferramentas horizontal, localizada acima da área de entrada de dados, apresenta os botões seguintes:

|   Botão    |   Objetivo                                                          |
|   ------    |  --------                                                          |
| **Guardar**    | Guarda as alterações de entrada de dados recentes.  Manualmente, deve salvar as alterações antes de navegar para fora da página ou as suas alterações serão perdidas. | 
| **Rejeição** | Descarta alterações recentes de entrada de dados (uma vez que o último guardar)             |
| **Comparar** | Compara o estado da oferta atual com a oferta publicada.  Só é ativada depois de uma oferta foi publicada com êxito.  |
| **Publicar** | Inicia o processo de publicação para esta oferta                       |
| **Eliminar**  | Depois de criado, mas antes de ele foi publicado, elimina esta oferta. |
|   |   |


## <a name="editing-tabs"></a>Separadores de edição

Ao criar uma oferta, fornecer os dados necessários e opcionais em cada separador localizado da coluna da esquerda vertical do **nova oferta** página.  Controles de interface de usuário padrão, como caixas de texto, menus de lista pendente e caixas de verificação, são apresentados para recolha de dados.  Embora a coleção de guias de edição específica depende do tipo de oferta, a tabela seguinte lista algumas das guias comuns.

|      Nome do separador       |   Objetivo                                                            |
|      --------       |   -------                                                            |
| **Definições da oferta**  | Recolhe a oferta e o publicador informações de identidade.                    |
| **SKU**            | Define as características técnicas e comerciais para cada versão de unidade de manutenção de estoque (SKU) da sua oferta |
| **Versão de teste**      | Para esses tipos que suportam esta funcionalidade opcional, define uma demonstração da sua oferta.  Para obter mais informações, consulte [o que é o Test Drive?](../../cloud-partner-portal-orig/what-is-test-drive.md)  |
| **Marketplace** ou **Vitrine** | Recolhe as cadeias de texto, documentos e imagens usadas para listar a oferta no marketplace |
| **Suporte**         | Recolhe informações de contacto para customer, engenharia e suporte online  |
|  |  |

O conteúdo das guias com o mesmo nome pode ser diferentes entre tipos de oferta diferente.  Detalhes de oferta específico dos seguintes separadores são fornecidos na secção "Criar oferta" para cada tipo de oferta.
