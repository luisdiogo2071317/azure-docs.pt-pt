A tabela seguinte inclui ligações para scripts do PowerShell do Azure de exemplo para o Azure Data Factory.

| |  |
|---|---|
|**Copiar dados**||
|[Copiar blobs a partir de uma pasta para outra pasta num armazenamento de Blobs do Azure](scripts/copy-azure-blob-powershell.md?toc=%2fpowershell%2fmodule%2ftoc.json)| Este script do PowerShell copia blobs a partir de uma pasta no armazenamento de Blobs do Azure para outra pasta num armazenamento de Blobs do mesmo. |

|[Cópia incremental](scripts/incremental-copy-powershell.md?toc=%2fpowershell%2fmodule%2ftoc.json)| Este script do PowerShell de exemplo carrega registos novos ou atualizados apenas de um arquivo de dados de origem para um arquivo de dados de sink após a cópia inicial completa dos dados da origem para o sink. |
