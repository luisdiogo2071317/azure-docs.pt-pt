# <a name="safely-install-and-operate-your-azure-data-box"></a>Instalar e operar o Azure Data Box em segurança
![Ícone de aviso](./media/data-box-safety/warning_icon.png)
![ícone de aviso de segurança de ler](./media/data-box-safety/read_safety_and_health_information_icon.png) **leitura informações de estado de funcionamento e segurança**

## <a name="handling-precautions"></a>Precauções de tratamento

![Ícone de aviso](./media/data-box-safety/warning_icon.png) ![ícone choque Elétrico](./media/data-box-safety/electrical_shock_hazard_icon.png)![nenhum utilizador serviceable na partes ícone](./media/data-box-safety/no_user_serviceable_parts_icon.png) **cuidado** 

* Inspecionar a *como recebido* dispositivo por danos. Se o bastidor de dispositivo está danificado, contacte o Support da Microsoft para obter uma substituição. Não tente operar o dispositivo. 
* O dispositivo é equipado com screws à prova. Se suspeitar que o dispositivo está a funcionar incorretamente, entre em contato com Support da Microsoft para obter uma substituição. Não tente para o dispositivo de serviço. 
* O dispositivo contém sem partes serviceable na utilizador. Tensão perigosa, atual e níveis de energia estão presentes no interior. Não abra. Devolva o dispositivo para a Microsoft para a manutenção.

![Ícone de aviso](./media/data-box-safety/warning_icon.png) ![ícone de peso pesada](./media/data-box-safety/heavy_weight_hazard_icon.png) **aviso!** 

* Um bastidor totalmente configurado pode avaliar até 22.7 kg (50 lbs); Não tente retirá-lo sozinho.
* Antes de passar o bastidor, certifique-se sempre de que duas pessoas estão disponíveis para lidar com o peso. Lembre-se de que uma pessoa tentar acabar com esta peso suportar injuries.


![Ícone de aviso](./media/data-box-safety/warning_icon.png) ![ícone de perigo de sugestão](./media/data-box-safety/tip_hazard_icon.png) **aviso!**
* Coloque o dispositivo numa superfície plana, disco rígida e estável para evitar uma potencial ameaça da sugestão.
* É montado em Bastidor equipamento não deve ser utilizada como prateleiras ou espaços de trabalho. Não coloque o Data Box com base no equipamento montado em bastidor. Adicionar qualquer tipo de carga a uma unidade montada em rack expandida, pode criar um risco de à sugestão potenciais que pode levar a lesão, morte ou danos de produto.

![Ícone de aviso](./media/data-box-safety/warning_icon.png) **aviso!**

* Configure o dispositivo numa área de trabalho, permitindo a circulação de ar adequado em torno do dispositivo.
* Instalar o dispositivo numa temperatura controlados fechada área livre de conductive contaminants e permitem circulação de ar adequado em torno do dispositivo.
* Manter o dispositivo para fora de fontes de ambientes de fluidos e excessivamente humid.


## <a name="electrical-precautions"></a>Precauções elétrica

![Ícone de aviso](./media/data-box-safety/warning_icon.png) ![ícone choque Elétrico](./media/data-box-safety/electrical_shock_hazard_icon.png) **aviso!**

* Fornece uma ligação de terra elétrico seguro para o cabo de alimentação de energia. O cabo de AC tem uma conexão de três durante a transmissão aterrados (uma conexão com um pin de aterrados). Este plug se encaixa apenas uma saída de AC mantermos. Não é contra o objetivo do pin aterrados.
* Dado que o plug no cabo de alimentação de energia é o principal desligar o dispositivo, certifique-se de que a comunicação de socket localização próxima ao dispositivo e são facilmente acessível.
* Desligue o cabo de alimentação (ao extrair plug, não, o cabo) e desligue todos os cabos, se existir alguma das seguintes condições:

    - O cabo de alimentação ou plug torna-se frayed ou caso contrário danificado.
    - Vazam-algo para as maiúsculas e minúsculas do dispositivo.
    - O dispositivo é exposto a chuva ou moisture em excesso.
    - O dispositivo foi removido e as maiúsculas e minúsculas de dispositivo está danificada.
    - Suspeitar de que o dispositivo tem de verificar ou reparar o serviço.
* Desligue permanentemente a unidade antes, movê-lo ou se achar que tem de se tornar danificado de qualquer forma.
* Forneça uma fonte de energia adequado com a proteção de sobrecarga elétrico para satisfazer as seguintes especificações de energia:

    - Tensão: AC de V 100 a 240 V AC
    - Atual: 6, máximo
    - Frequência: 50 Hz a 60 Hz

![Ícone de aviso](./media/data-box-safety/warning_icon.png) **cuidado:**

* Este dispositivo contém baterias de célula de moeda. Não tente para o dispositivo de serviço. Baterias neste dispositivo não for utilizador serviceable na. 
* **Para o pessoal do serviço apenas**: risco de explosão se bateria é substituída por um tipo incorreto. Descarte as baterias utilizadas, de acordo com as instruções.

![Observe o ícone](./media/data-box-safety/notice_icon.png) **aviso:**

Para a operação adequada do seu dispositivo e para evitar danos de produto:

* Certifique-se de que o front-end e portas abertas são totalmente abertas enquanto o dispositivo está em execução.

## <a name="regulatory-information"></a>Informações de regulamentação

Esta secção contém informações normativas para o Azure Data Box, o número de modelo regulamentares DB010.

Este dispositivo é:

- Avaliada como informações de tecnologia de equipamentos (ITE), concebido para operar num ambiente de sala de típica de dados. A adequação do produto para outros ambientes pode exigir mais avaliação.
- Concebido para utilização com NRTL listados (UL, CSA, ETL, etc.) e IEC/EN 60950-1 ou IEC/EN 62368-1 em conformidade (CE marcado) equipamento de tecnologia da informação.
- Concebido para operar no ambiente seguinte. 
    - Operar a temperatura: 50° para 95° F (10° 35° c)
    - Temperatura de armazenamento:-4 ° para 122 ° F (-20 ° 50 ° c)
    - Humidade relativa: 15% a 85% (noncondensing) 
    - Operacional em altitude: testado até 6500 pés (metros de 0 a medidores de 2000)

Para as classificações de fonte de alimentação elétrica, consulte a etiqueta de classificação de dispositivo fornecida com a unidade.
