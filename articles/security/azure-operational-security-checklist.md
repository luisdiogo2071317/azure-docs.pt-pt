# <a name="azure-operational-security-checklist"></a>Lista de verificação de segurança operacionais do Azure
Implementar uma aplicação no Azure é rápida, fácil e económica. Antes de implementar a aplicação na cloud em produção útil ter uma lista de verificação para ajudar a avaliar o seu aplicativo numa lista de ações de essenciais e recomendados de segurança operacional a ser considerada.

## <a name="introduction"></a>Introdução

O Azure fornece um conjunto de serviços de infraestrutura que pode utilizar para implementar as suas aplicações. Segurança operacional do Azure refere-se a serviços, controles e recursos disponíveis aos utilizadores para proteger seus dados, aplicações e outros ativos no Microsoft Azure.

-   Para obter o máximo benefício fora a plataforma na cloud, recomendamos que tire partido de serviços do Azure e siga a lista de verificação.
-   As organizações que investir tempo e recursos de avaliação da prontidão operacional das suas aplicações antes de iniciar ter uma taxa muito maior de satisfação do que aqueles que não. Ao realizar esse trabalho, listas de verificação podem ser um mecanismo inestimável para garantir que os aplicativos são avaliados consistentemente e holística.
-   O nível de avaliação operacional será varia dependendo do nível de maturidade da organização na cloud e a fase de desenvolvimento do aplicativo, as necessidades de disponibilidade e requisitos de confidencialidade de dados.

Também pode ser utilizado para ajudar a criar uma estratégia de migração e a operação de cloud segura para a sua organização.
