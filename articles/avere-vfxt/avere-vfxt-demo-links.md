# <a name="avere-vfxt-demo-projects"></a>Projetos de demonstração de vFXT Avere

Tutoriais de exemplo são fornecidas numa [GitHub](https://github.com/Azure/Avere). Esses projetos pequenos demonstram recursos principais e casos de utilização para Avere vFXT para o Azure. 

## <a name="video-rendering"></a>Processamento de vídeo

* [Composição com o Azure Batch e Avere vFXT](https://github.com/Azure/Avere/blob/master/docs/maya_azure_batch_avere_vfxt_demo.md) -uma demonstração de 60 minutos, que demonstra como utilizar o Autodesk Maya com o Azure Batch e de um cluster de vFXT Avere para gerar um filme animado

* [Porquê utilizar o vFXT Avere para composição?](https://github.com/Azure/Avere/blob/master/docs/why_avere_for_rendering.md) -Uma demonstração que compara os tempos de processamento de com e sem um cluster de vFXT Avere 


## <a name="high-performance-computing"></a>Computação de alto desempenho

* [Melhores práticas para o tempo de inicialização de melhorar a Máquina Virtual (VM) do Azure](https://github.com/Azure/Avere/blob/master/docs/azure_vm_provision_best_practices.md) -um teste reproduzível da utilização Avere vFXT para o Azure para melhorar a inicialização vezes quando Acelere rapidamente a milhares de clientes de computação

## <a name="vfxt-performance"></a>desempenho de vFXT 

* [Medir o desempenho de vFXT com vdbench](https://github.com/Azure/Avere/blob/master/docs/vdbench.md) -uma configuração de teste básico para gerar as cargas de trabalho de pequenas e médio porte para testar os subsistemas de memória e disco vFXT

## <a name="client-setup"></a>Configuração do cliente

* [Estação de trabalho do Windows 10 para Avere vFXT](https://github.com/Azure/Avere/blob/master/docs/windows_10_avere_vfxt_mounted_workstation.md) -demonstra como configurar uma estação de trabalho do Windows e montá-la a um cluster de vFXT Avere
